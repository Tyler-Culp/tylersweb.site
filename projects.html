  1. Android Compass App (link)

    This was a quarter long project that I worked on with a team of 6 people consisting of two app iterations. The first was an app where you could input 3 different coordinate (assumedly corresponding to your house, your parent's house, and a friend's house) and 3 unique labels and then see icons of those house including your labels on a live compass. This compass would update as you turned your phone or moved locations, meaning that the icons corresponding to each set of coordinates would always be pointing in the same direction. In the second iteration instead of inputing coordinate, you inputted friends unique ID's to be able to pull their current live coordinates from a database. This version of the compass would update as you turned your phone or moved around, or as your friend's moved around. Both applications were fully built in Android Studio using mainly Java.

  2. Marvel Movie Success Predictor

    In this project I worked with a team of 5 other people where we tried to discover if there was a way to predict the success of a marvel movie, both critically and commercially, based on general movie statistics like its run time, budget, year released, etc. This was mainly done with python inside of Jupyter Notebooks where we used libraries like pandas, numpy, patsy, statsmodels, etc. Through this project we found that things like production budget did have a positive correlation with the success of the movie.

  4. Nachos (Operating System)

    Using the nachOS software and in a team of 4 people we built up a simple operating system by including features like threading, system calls, paging / on demand paging, and swapping. During the process we made heavy use of things like locks and condition variables to ensure mutual exclusion where necessary and to try and keep our basic OS as secure as we could possible make it. Through this project I gained a deeper understanding of threading and how multithreaded programs work, how a standard operating system works (as well as the main design principles for them), and security principles to adhere to when building any kind of OS.
